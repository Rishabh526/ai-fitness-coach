# AI Fitness Coach

AI-Fitness-Coach is a full-stack application designed to provide personalized fitness guidance using Generative AI.

## Current Status
- Backend authentication implemented using Django and JWT
- User registration and login APIs completed

## Tech Stack
- Backend: Django, Django REST Framework, JWT
- Frontend: React (planned)
- AI: Generative AI (planned)

## Project Structure
AI-Fitness-Coach/
├── backend/ # Django backend
├── frontend/ # React frontend (coming soon)
└── README.md

## Roadmap
- Connect authentication with React frontend
- User onboarding (fitness profile)
- AI-powered workout and diet recommendations
